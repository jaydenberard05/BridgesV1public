# Bridges OS v 1.0.0

 Bridges OS is a mini operating system designed using C# and COSMOS (C# Open Source Managed Operating System)

### Features in this OS include
-A File System (FAT32)

-Calculator

-Settings Menu

-About Menu

-Very small DnD demo 

-Startup music (does not work with VMWare vitural machine)
